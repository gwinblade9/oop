Система доставки еды
Веб-система для управления заказами в службе доставки.

Что умеет:
- Создавать заказы из меню ресторанов
- Менять статусы: Новый → Принят → Готовится → Готов → В пути → Доставлен
- Отменять заказы (только на первых двух этапах)
- Смотреть аналитику: количество заказов, выручка, среднее время

Запуск:
bash
# Установить зависимости
pip install -r requirements.txt

# Запустить
python run_web.py
Открыть в браузере: http://127.0.0.1:5000

Структура:
text
src/          # Классы и работа с БД
web/          # Flask, шаблоны, стили
data/         # SQLite база (создаётся сама)

Тестовые данные
При первом запуске создаются:
- 3 пользователя (Иван, Мария, Алексей)
- 3 ресторана с меню
- 2 тестовых заказа

Если что-то сломалось
Удалить data/delivery.db и перезапустить — база создастся заново.

Зависимости:
text
Flask==2.3.3
Flask-CORS==4.0.0
